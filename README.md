# Hongzhi Lu / 卢鸿志 Academic Website

This package is ready for GitHub Pages deployment.

Files:
- index.html: compact bilingual academic homepage with one-line Hongzhi Lu title and profile photo.
- profile-photo.jpg: profile image used in the homepage dashboard.
- eco-scan-monster.html: Eco Scan Monster interactive software demo.
- human-battery-baby.html: Human Battery Baby interactive software demo.
- CV.pdf: downloadable CV.

Deploy:
1. Upload all files to the root of your GitHub Pages repository.
2. Go to Settings -> Pages.
3. Select Deploy from branch -> main -> /root.
4. Save and wait for GitHub Pages to publish.
